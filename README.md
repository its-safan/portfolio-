# portfolio-
Personal portfolio website built with HTML, CSS &amp; JavaScript. Features 3D animations, particle effects, and a dark futuristic design.
📖 About

This is my personal developer portfolio — designed to stand out. Built entirely from scratch using vanilla HTML, CSS, and JavaScript, with no external libraries or frameworks. It showcases my projects, skills, and contact information inside a fully interactive, dark futuristic UI.

The goal was to prove that you don't need React or complex tooling to build something immersive and high-performance.


✨ Features


Animated Loader — Smooth percentage-counting intro screen before the site reveals
Interactive Particle Canvas — 90-node WebGL-style particle network that reacts to mouse movement
Custom Neon Cursor — Glowing dot with a lagging ring that follows your mouse
Glitch Text Effect — Hero headline with a CSS-only RGB glitch animation
3D Hero Scene — Orbiting rings, floating cubes, and an animated sphere built in pure CSS
Scroll-Triggered Animations — Elements fade and slide in as you scroll using Intersection Observer
Animated Stat Counters — Numbers count up from zero on page load
3D Card Tilt — Every card rotates in 3D following your mouse position
Skill Progress Bars — Bars animate into view when the section scrolls into frame
Live Project Canvas Scenes — Each project card has its own animated Canvas mini-scene (grid, code lines, wireframe cube)
Working Contact Form — With animated send state and success confirmation
Fully Responsive — Works on mobile, tablet, and desktop
Zero Dependencies — No npm, no bundler, no frameworks — open and run
